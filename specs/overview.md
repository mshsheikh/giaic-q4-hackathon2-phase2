# Todo Web App - Project Overview

## Purpose
This project is a multi-user Todo web application that allows users to create, manage, and organize their personal tasks in a secure and intuitive interface. The application provides user authentication, task management, and data isolation between users.

## Current Phase
Phase II: Full-stack web application with comprehensive specifications, architecture, authentication model, API contracts, and database schema defined using Spec-Kit Plus methodology. This phase focuses on creating detailed specifications without implementation.

## Tech Stack
- **Frontend**: Next.js 14+ with App Router and React Server Components
- **Backend**: FastAPI (Python 3.11+) with async support
- **Database**: SQLModel with Neon PostgreSQL
- **Authentication**: Better Auth for JWT-based authentication
- **Architecture**: Monorepo structure
- **API Style**: RESTful with JSON responses
- **Security**: JWT tokens, HTTPS, input validation, and data isolation
